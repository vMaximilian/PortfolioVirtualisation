NSX-T Datacenter

Network virtualisation and security platform that looks at provisioning and managing networks independently of the underlying hardware.

Pain point and how we solve it
Pain point

Network is usually a siloed environment from within datacenters. It needs specific hardware and trained workforce, which represents expensive investments and vendor lock-in. On the security point of view, traditionally the security is seen as something perimetral.

Solution

Create multiple virtual networks with diverse requirements. Deploy firewalls around each virtual machine.
