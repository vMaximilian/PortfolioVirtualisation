VMware Cloud on AWS

Enables customers to run, manage, and secure applications in a production ready, simple and consistent hybrid IT environment with access to innovative AWS services. Use exisitng VMware tools to manage VMware Cloud on AWS environments without having to purchase custom hardware, rewrite applications, or modify their operating model. The service provides access across the broad range of AWS services with the functionality, elasticity, and security customers have come to expect from the AWS Cloud.

Pain point and how we solve it
Pain point

DC hardware coming to EOL/EOS, datacenter lacking capacity and flexibility. Wanting to keep Vmware environments while exploring cloud capabilities. Also app modernization and use of cloud services.

Solution

Co-engineered solution from Vmware and AWS to provide customers the possibility to start deploying workloads in the cloud with the same tools they use on premise and move them between sites without downtime, enabling the true hybrid cloud. It comes with all SDDC Vmware components (compute, storage, networking and security) and provides easier disaster protection, AWS integration and access to several regions.
